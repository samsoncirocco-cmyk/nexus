# Roadmap

## Phase 1: Real-time Text Intelligence
- Gmail ingestion → Pub/Sub → BigQuery
- NLP enrichment (entities + sentiment)
- Minimal query surface

## Phase 2: Omnimodal Memory
- Drive ingestion for images/audio/video
- Vision, Speech-to-Text, Video Intelligence enrichment

## Phase 3: Spatial Intelligence
- Geocode addresses and infer locations
- Places enrichment for context

## Phase 4: Predictive Intelligence
- BigQuery ML models for habit and workload prediction
- Vertex AI Search for unified semantic search

## Phase 5: Proactive Agents
- Task triage and scheduling
- Automated follow-ups with approval gates
