# 3-Layer Architecture

Standard framework across all repos:

1. **Directives** — High-level instructions, SOPs, decision rules
2. **Orchestration** — Coordination logic, routing, scheduling  
3. **Execution** — Scripts, tools, API calls that do the actual work

Applied to: TatT, Second Brain, Nexus, Book Toolkit, IRL, SFDC
Template repos: SFDC and Project Apex
