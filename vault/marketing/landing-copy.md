## FAQ

### 1. What is Second Brain exactly?
Second Brain is a web-based dashboard that sits on top of your AI agent infrastructure. It gives you real-time visibility into what your agents are doing, lets you chat with them, search your knowledge base, track costs, and manage tasks â€” all in one beautiful interface.

### 2. Do I need to use OpenClaw?
The full agent management features work best with OpenClaw as the backend, but the knowledge vault, task board, sales pipeline, and document viewer work standalone. You can start with just the knowledge base and add agent features later.

### 3. Is my data stored in the cloud?
Your knowledge vault is stored as plain Markdown files â€” you own them completely. The app runs on Vercel but your vault data stays in your repository. No vendor lock-in, no proprietary formats.

### 4. How much does it cost?
Second Brain itself is free and open source (MIT license). You'll need API keys for the AI features (Google Gemini for Q&A, plus whatever models your agents use). Hosting on Vercel's free tier works for personal use.

### 5. Can I use it on mobile?
Yes! Second Brain is a Progressive Web App (PWA). Install it on your phone's home screen for offline access, push notifications, and a native-feeling experience. Works on iOS and Android.
