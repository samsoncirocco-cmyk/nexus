# TatT

AI-powered tattoo design platform. The full pipeline: Generate → Preview → Match → Book.

## Stack
- Next.js + Tailwind CSS
- Replicate (AI image generation)
- Firebase (matching/auth)
- Supabase (vector DB)
- Vercel deployment

## Status
